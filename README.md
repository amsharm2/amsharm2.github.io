# Amit's Homepage - Data Engineering & Analytics Portfolio

Welcome to the source code for my personal portfolio website. This site showcases my professional journey, technical expertise, and projects in Data Engineering, Analytics, and AI.

This project is a static website built with [Jekyll](https://jekyllrb.com/), hosted on GitHub Pages. It serves as a digital resume and portfolio, highlighting my experience as a Data Professional with over 19 years of experience in building high-impact data solutions.

**Key Highlights:**
*   **Role:** Data Engineering | Analytics Engineering | Data Analytics | Mentor
*   **Experience:** 19+ years across Quick Commerce, Fintech, and Retail domains.
*   **Focus:** Driving Business Value, Strategic Architecture, Cost Optimization, Team Leadership, and Innovation.
*   **Tech Stack:** SQL, Python, dbt, Airflow, Snowflake, Databricks, and more.

## Setup Instructions

To run this site locally, you'll need to have Ruby and Bundler installed on your machine.

### Prerequisites

*   **Ruby**: Ensure you have a compatible version of Ruby installed.
*   **Bundler**: Install bundler using `gem install bundler`.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/amsharm2/amsharm2.github.io.git
    cd amsharm2.github.io
    ```

2.  Install dependencies:
    ```bash
    bundle install
    ```

### Usage

1.  Build and serve the site:
    ```bash
    bundle exec jekyll serve
    ```

2.  Open your browser and navigate to `http://localhost:4000` to view the site.
